# Priyanka's Website

This is the repository for Priyanka's personal website. Below are the details regarding the project structure and setup instructions.

## Project Structure

```
priyanka-website
├── index.html          # Main HTML document for the website
├── style.css           # CSS styles for the website
├── assets              # Folder for all media files
│   ├── fonts           # Folder for font files
│   └── media           # Folder for media files (images, videos, etc.)
├── resources           # Optional folder for future resource lists
│   └── links.json      # JSON file for storing links in a structured format
└── README.md           # Documentation about the project
```

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the website.
3. Modify the `style.css` file to customize the appearance of the website.
4. Add any media files to the `assets/media` folder.
5. Update the `resources/links.json` file as needed to manage your links.
